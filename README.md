# smartmeterapi

This Rest API is built using **Spring** and **H2** in memory database.   
You can run this REST service by command:  
**mvn spring-boot:run**

Swagger documentation can be found at:  
**/swagger-ui-custom.html**

H2 console can be found at:  
**/h2-console**
